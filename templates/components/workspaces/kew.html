📦 Xử lý KEW (hồ sơ hiện trường)

Upload một file .zip gồm: file Excel hiện trường (cột Tên thiết bị, File, IMG, IMG end), các thư mục record dạng S0001, S0002, … và ảnh PS-SDxxx.BMP. Hệ thống đổi tên thư mục theo tên thiết bị, chuyển ảnh vào đúng thư mục tương ứng, kiểm tra thiếu file / trùng dải ảnh, rồi trả về file KEW_HoSoDaXuLy.zip (cấu trúc Project_Output/…).

📝 Tạo báo cáo Word

Upload file KEW_HoSoDaXuLy.zip (output của bước "Xử lý file sơ bộ"), hoặc một ZIP chứa các thư mục thiết bị có INPSxxxx.KEW + ảnh PS-SDxxx.BMP. Hệ thống tổng hợp số liệu (U, I, P, Q, S, PF, THD, TDD…), dựng báo cáo từ template MBA / device và trả về BaoCao_KEW.docx. Nếu ZIP có kèm Excel với các cột tuỳ chọn Loại, Điện áp định mức, Nhận xét thì các giá trị này được dùng làm metadata.
