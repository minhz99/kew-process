📦 Xử lý KEW (hồ sơ hiện trường)

Upload một file .zip gồm: file Excel hiện trường .xlsx với đúng các cột stt, name, file, img, imgend, imgomit, type, pdm, p, pf, i1, i2, i3, di, thd, tdd (tên cột không phân biệt hoa thường), các thư mục record S0001, S0002, … và ảnh PS-SDxxx.BMP. Bước này dùng name / file / img / imgend / imgomit (tuỳ chọn: liệt kê chỉ số ảnh trong dải bị loại, vd 944,945); stt và các cột còn lại phục vụ bước tạo Word (thứ tự thiết bị, loại MBA/device, …). Hệ thống đổi tên thư mục theo tên thiết bị, chuyển ảnh vào đúng thư mục, kiểm tra thiếu file / trùng dải ảnh, rồi trả về KEW_HoSoDaXuLy.zip (cấu trúc Project_Output/…).

📝 Tạo báo cáo Word

Upload file KEW_HoSoDaXuLy.zip (output của bước "Xử lý file sơ bộ"), hoặc ZIP chứa các thư mục thiết bị có INPSxxxx.KEW + ảnh PS-SDxxx.BMP. Hệ thống tổng hợp số liệu từ INPS, dựng báo cáo Word (BaoCao_KEW.docx). Nếu ZIP có kèm Excel .xlsx đủ bộ cột hiện trường (cùng mẫu với bước «Xử lý file sơ bộ»), hệ thống đọc stt (thứ tự thiết bị trong báo cáo), type, pdm (điện áp định mức), p, pf, i1–i3, di, thd, tdd làm metadata (loại section, điện áp định mức, chuỗi nhận xét ghép từ các chỉ số).
